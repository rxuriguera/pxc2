Estadísticas

¿Qué es?

És una aplicación que permite escuchar música vía streaming y compartir música mediante peer to peer.

Primeros pasos

Para poder utilizar Speerker es necesario crear una cuenta de usuario. Para ello, debemos acceder al apartado Crear Cuenta.

Crear Cuenta

Debemos rellenar los campos del formulario y acceptar. Una vez finalizado el registro, ya podremos acceder como usuario registrado.

Acceder Como Usuario Registrado

En la parte izquierda de la página principal hay un apartado de acceso a usuarios registrados. Debemos entrar con nuestro nombre de usuario y contraseña.

¿Cómo utilizar Speerker?

  • Una vez hemos entrado con nuestro usuario registrado debemos descargarnos la aplicación.
  • Después de instalarla ya podemos empezar a utilizar Speerker.
  • Desde la aplicación podemos buscar, escuchar y comporatir música.
  • Tanto desde la aplicación como desde la web podemos consultar las estadísticas de la aplicación.
